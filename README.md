# 📚 AI Study Planner (Flask Project)

## 🚀 Overview
This is a smart AI-based Study Planner web application built using Flask (Python).
It helps students generate a personalized study plan based on inputs like subject, hours, days, difficulty, and goals.

## 🎯 Features
- Predicts study performance
- Generates study plan
- Simple UI
- Fast backend

## 🛠️ Tech Stack
- HTML, CSS, JavaScript
- Python (Flask)

## ⚙️ How to Run

1. Clone repo:
git clone https://github.com/Er-Princegit/Python-Flask-Project.git

2. Go to folder:
cd Python-Flask-Project

3. Install Flask:
pip install flask

4. Run app:
python app.py

5. Open:
http://127.0.0.1:5000/
